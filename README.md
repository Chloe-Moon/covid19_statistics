# covid19_statistics

네이버에 '코로나 현황' 페이지를 크롤링하여 코로나 현황을 분석 및 시각화 한다. 

## 카카오 지도 API 사용 방법
- https://developers.kakao.com/ 에 접속하여 로그인(회원가입)한다. 
- 상단 메뉴 바에 <내 애플리케이션>을 눌러 <애플리케이션 추가하기>를 한다. 
- REST API 키를 복사하여 사용한다.
